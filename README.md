# Java Spring Boot JPA

# Sobre o projeto

Este projeto é um back-end simples feito em Java usando Spring Boot e JPA, construído em acompanhamento do curso de Java organizado pelo professor [Nelio Alves](https://www.udemy.com/course/java-curso-completo/?audience=Keyword&campaigntype=Search&funnel=Conversion&language=PT&matchtype=b&portfolio=Brazil&priority=Beta&product=Course&test=&topic=Java&couponCode=NEWYEARCAREER#instructor-1).

A aplicação consiste em uma criação de usuários, categorias, produtos e pedidos parecidos com o que se encontra em sistemas de gerenciamento de produtos e compras.


# Tecnologias utilizadas
- Java
- Spring Boot
- JPA / Hibernate
- Maven

## Implantação em produção
- Banco de dados: Postgresql

# Como executar o projeto

## Back end
Pré-requisitos: Java 17 e Maven

```bash
# clonar repositório
git clone https://github.com/p3dr0dev/workshop-springboot-jpa

# entrar na pasta do projeto back end
cd workshop-springboot-jpa

# executar o projeto
./mvn spring-boot:run
```
